# Billing – TryHackMe Writeup

Bu makinede web uygulamasındaki zafiyeti kullanarak sisteme erişim sağladım.

## Kullanılan araçlar

- Nmap
- Gobuster
- Searchsploit

## Yapılanlar

- Nmap ile servisleri taradım
- Web servisinin çalıştığını tespit ettim
- Gobuster ile admin panelini buldum
- Servis versiyonunu tespit ettim
- Searchsploit ile exploit araştırdım
- Exploit kullanarak sisteme eriştim

## Öğrendiklerim

- Servis versiyon analizi yapmayı
- Exploit araştırma sürecini
- Web exploitation mantığını
