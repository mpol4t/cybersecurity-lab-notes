# Pickle Rick – TryHackMe Writeup

Bu makinede web sitesi üzerinden komut çalıştırarak sisteme erişim sağladım.

## Kullanılan araçlar

- Nmap
- Web browser
- Linux terminal

## Yapılanlar

- Nmap ile açık portları tespit ettim
- Web sitesini inceledim
- Kaynak kodlarını kontrol ettim
- Login bilgilerini elde ettim
- Web panel üzerinden komut çalıştırarak sisteme eriştim

## Öğrendiklerim

- Web enumeration sürecini
- Kaynak kod analizinin önemini
- Web üzerinden komut çalıştırmayı
