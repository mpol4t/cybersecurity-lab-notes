# RootMe – TryHackMe Writeup

Bu makinede web enumeration, file upload vulnerability ve privilege escalation pratiği yaptım.

## Kullanılan araçlar

- Nmap
- Gobuster
- Netcat
- Linux terminal

## Yapılanlar

- Nmap ile açık portları tespit ettim
- Gobuster ile gizli dizinleri buldum
- File upload vulnerability üzerinden reverse shell aldım
- Privilege escalation ile root erişimi sağladım

## Öğrendiklerim

- Enumeration sürecinin önemini
- Web zafiyetlerinin nasıl exploit edildiğini
- Reverse shell almayı
- Privilege escalation mantığını
