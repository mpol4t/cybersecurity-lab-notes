# Cybersecurity Lab Notes

Bu repository, siber güvenlik öğrenme sürecimde çözdüğüm CTF makinelerinin writeup'larını ve kullandığım araçlarla ilgili kişisel notlarımı içermektedir.

Amacım, penetration testing ve system exploitation konularında pratik yaparak teknik yetkinliğimi geliştirmek ve öğrendiklerimi dokümante etmektir.

---

## İçerik

### Machines

Bu klasörde çözdüğüm CTF makinelerinin çözüm notları bulunmaktadır.

Her writeup şu aşamaları içermektedir:

- Reconnaissance (Nmap ile port tarama)
- Enumeration (Gobuster ile dizin keşfi, servis analizi)
- Exploitation (zafiyetin exploit edilmesi)
- Privilege Escalation (root yetkisi elde etme)
- Öğrenilen tekniklerin özeti

Örnek makineler:

- RootMe
- Billing
- Responder
- Pickle Rick
- Vulnversity
- Three
