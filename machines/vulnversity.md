# Vulnversity – TryHackMe Writeup

Bu makinede web enumeration ve file upload zafiyeti üzerinden sisteme erişim sağladım.

## Kullanılan araçlar

- Nmap
- Gobuster
- Burp Suite
- Netcat

## Yapılanlar

- Nmap ile açık portları taradım
- Gobuster ile gizli dizinleri buldum
- Upload paneli olduğunu tespit ettim
- Zararlı payload yükleyerek reverse shell aldım
- Sistemde gerekli dosyaları inceleyerek erişimi ilerlettim

## Öğrendiklerim

- Web enumeration sürecini
- Gobuster kullanımını
- File upload zafiyetlerinin nasıl exploit edildiğini
- Reverse shell almayı
